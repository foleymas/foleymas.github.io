Introduction to Operating Systems

Disk Scheduler

Implemented a basic multi threaded disk scheduler. Running a thread for each disk to schedule requests and a main scheduler thread. Handled requests based on closest seek time first and allowed for a varriable number of cpus.

Thread Library

Implemented a light thread library for a multi-cpu system. This included cpu interrupt handling for timer interrupts and interrprocess interupts. Thread functions such as thread construction, yield and join. Mutex functions such as lock and unlock. CV functions such as wait, signal and broadcast. All blocking was done on blocking queues using FIFO ordering.

Pager

Implemented a vitual memory pager. Functionality included swap backed and file backed virtual pages, copy on write protection for shared virtual pages, virtual page initialization with non empty arenas.

Network File Server

Implemented the server side a a network file server application. Supported creation of new files or directories, deletion of files or directories, reading from files, and writing to files. Server data protected using reader-writer locking.
